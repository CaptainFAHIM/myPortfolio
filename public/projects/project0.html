Features :

  • Implemented two (customer, admin) role.
  • Customers can add pizza to the cart and place an order by providing their address and phone number.
  • Admins will receive the order with details and can set real-time status for it.
  • Customers can check the status of their order by clicking on the order ID.
  • Customers can also view their previous order history.
  • All the data is being stored in MongoDB.

Tech stack :
